
# План: Добавить выбор модуля-папки для уроков в Мастере добавления контента

## ✅ ВЫПОЛНЕНО

### Изменения

1. **Создан `ModuleSelector.tsx`** — компонент для выбора целевого модуля:
   - Опция "Отдельный урок" (контейнер, как раньше)
   - Список существующих модулей в разделе (is_container=false)
   - Быстрое создание нового модуля inline

2. **Обновлён `ContentCreationWizard.tsx`**:
   - Добавлено поле `targetModuleId: string | null` в WizardData
   - Добавлен новый шаг "Модуль" (step 2) в lesson flow
   - Обновлены LESSON_STEPS: теперь 6 шагов вместо 5
   - Логика создания урока использует выбранный модуль или контейнер

### Новый flow для уроков

```
Раздел → Тип → Модуль → Доступ → Урок → Готово
  (0)    (1)    (2)      (3)     (4)     (5)
```

### DoD

| # | Проверка | Ожидание |
|---|----------|----------|
| 1 | Открыть wizard → Урок | После шага "Тип" появляется шаг "Модуль" |
