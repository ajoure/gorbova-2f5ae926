

# План: Описание урока — полное, под видео, красиво оформлено

## Текущее состояние
- Описание находится в секции "Lesson Header" (строки 226–269)
- На desktop: обрезается (`line-clamp-2`) с HoverCard для полного текста
- На mobile: обрезается с Collapsible "Показать полностью"
- Всё это **над** видео

## Что изменим
Переносим описание **под видео** и делаем его **всегда полным**, красиво оформленным.

---

## Изменения в `src/pages/LibraryLesson.tsx`

### 1. Убрать описание из шапки (строки 226–269)
Удаляем весь блок с описанием из секции "Lesson Header".

### 2. Добавить описание после видео-контента
После блоков/видео (после `{/* Block-based Content ... */}`) и **перед** "Attachments" добавляем новую карточку с описанием:

```text
Структура:

┌─────────────────────────────────────┐
│ Шапка: заголовок + бейджи           │
│ (без описания)                      │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│ Видеоплеер                          │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│ 📋 Описание выпуска                  │
│                                     │
│ Полный текст описания...            │
│ (красиво оформлен, prose стили)     │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│ Вопросы этого выпуска               │
└─────────────────────────────────────┘
```

### 3. Оформление описания

```tsx
{/* Описание выпуска — под видео, полный текст */}
{currentLesson.description && (
  <Card className="mb-6">
    <CardHeader className="pb-3">
      <CardTitle className="text-lg flex items-center gap-2">
        <FileText className="h-5 w-5" />
        Описание выпуска
      </CardTitle>
    </CardHeader>
    <CardContent>
      <p className="text-foreground whitespace-pre-wrap leading-relaxed">
        {currentLesson.description}
      </p>
    </CardContent>
  </Card>
)}
```

**Стилизация:**
- `whitespace-pre-wrap` — сохраняет переносы строк из текста
- `leading-relaxed` — увеличенный межстрочный интервал для читаемости
- `text-foreground` — основной цвет текста (не приглушённый)
- Карточка с иконкой и заголовком — визуально выделяет секцию
